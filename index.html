Tutorial Python

Fecha, Hora
y Nombre

Aprende paso a paso cómo crear tu primer programa en Python que captura el tiempo actual y saluda al usuario por su nombre.

⏱ 10 min · 🐍 Python 3.x · 🟢 Principiante
Comenzar →

Antes de empezar

🐍 Python instalado

Versión 3.6 o superior. Descárgalo en python.org

📁 Editor de código

Visual Studio Code, PyCharm o cualquier editor de texto.

💻 Terminal

CMD, PowerShell (Windows) o Terminal (Mac/Linux).

01

Crea el archivo del proyecto

Abre tu editor de código y crea un nuevo archivo. Guárdalo con el nombre programa.py en una carpeta de tu elección, por ejemplo C:\MisProgramas\.

💡 Tip La extensión .py le indica a tu sistema que es un script de Python.
02

Importa el módulo datetime

Python incluye de forma nativa el módulo datetime, que nos permite trabajar con fechas y horas sin instalar nada extra.

Escribe esto al inicio de tu archivo programa.py:

programa.py 
from datetime import datetime

¿Qué hace esto? Importa la clase datetime desde el módulo datetime de Python. Esta clase contiene los métodos para obtener la fecha y hora actual.

03

Obtén la fecha y hora actual

Usa el método .now() para capturar el momento exacto en que se ejecuta el programa:

programa.py 
from datetime import datetime

ahora = datetime.now()
print("📅 Fecha actual:", ahora.strftime("%d/%m/%Y"))
print("🕐 Hora actual: ", ahora.strftime("%H:%M:%S"))
CódigoSignificado
datetime.now()Captura el instante actual del sistema.
strftime("%d/%m/%Y")Formatea: día/mes/año (ej. 11/03/2026)
strftime("%H:%M:%S")Formatea: hora:minutos:segundos (ej. 14:35:07)
04

Pide el nombre al usuario

Usamos la función nativa input() para solicitar datos al usuario desde la terminal. El texto ingresado se almacena en una variable:

programa.py 
nombre = input("👤 ¿Cuál es tu nombre? ")
💡 Tip La variable nombre guarda en memoria lo que el usuario escriba. Puedes reutilizarla las veces que quieras dentro del programa.
05

Guarda el nombre en un archivo .txt

Para guardar el nombre de forma permanente (que no se pierda al cerrar el programa), lo escribimos en un archivo de texto usando open():

programa.py 
with open("usuario.txt", "w", encoding="utf-8") as archivo:
    archivo.write(f"Nombre: {nombre}\n")
    archivo.write(f"Registrado el: {ahora.strftime("%d/%m/%Y a las %H:%M:%S")}\n")

print(f"✅ ¡Hola, {nombre}! Tu nombre fue guardado correctamente.")
ParámetroSignificado
"usuario.txt"Nombre del archivo a crear/sobreescribir.
"w"Modo escritura. Crea el archivo si no existe.
encoding="utf-8"Soporta tildes y caracteres especiales.
f"..."f-string: permite insertar variables dentro del texto.
06

Código completo

Este es el programa final con todo integrado. Cópialo en tu archivo programa.py:

programa.py — Versión final ✅ 
# ─────────────────────────────────────
# Programa: Fecha, Hora y Nombre
# Autor: Tu Nombre
# ─────────────────────────────────────

from datetime import datetime

# 1. Capturar fecha y hora actual
ahora = datetime.now()
print("📅 Fecha actual:", ahora.strftime("%d/%m/%Y"))
print("🕐 Hora actual: ", ahora.strftime("%H:%M:%S"))
print()

# 2. Solicitar nombre al usuario
nombre = input("👤 ¿Cuál es tu nombre? ")

# 3. Guardar nombre en archivo
with open("usuario.txt", "w", encoding="utf-8") as archivo:
    archivo.write(f"Nombre: {nombre}\n")
    archivo.write(f"Registrado el: {ahora.strftime('%d/%m/%Y a las %H:%M:%S')}\n")

# 4. Confirmar al usuario
print(f"\n✅ ¡Hola, {nombre}! Tu nombre fue guardado en 'usuario.txt'.")
07

Ejecuta el programa

Abre tu terminal, navega a la carpeta donde guardaste programa.py y ejecuta:

Terminal / CMD 
$ python programa.py
